# Intelligent field audit

ApiaryLens uses controlled choices when the domain is finite and a suggestion plus
custom-value escape hatch when beekeeping practices, products, regulations, units,
or local terminology vary. Required status, role, synchronization, and Boolean
observations remain closed choices. Names, notes, exact label text, and local
identifiers remain user-owned text.

| Workflow | Field behavior | Reason |
|---|---|---|
| Queen | Identifier kind, mark mode/color/year, and source choices with Other | International marking cycle plus local identifiers |
| Equipment | Component type and purpose choices; numeric frames; ordered stack | Physical configuration needs consistent visualization |
| Inspection | Closed temperament/population/yes-no choices; suggested brood/stores/conditions; numeric weather | Comparable history with room for observation detail |
| Mite count | Closed method plus validated sample/count | Percent can be derived consistently |
| Health | Suggested common concerns plus custom text and explicit non-diagnosis copy | Regional conditions vary and the product must not imply a diagnosis |
| Feeding | Suggested type, unit, and reason plus recent/custom values | Family recipes and units vary |
| Treatment | Suggested method, recent exact values, label/restriction reminders, custom text | Products and law vary; recordkeeping must not become medical/legal advice |
| Harvest | Numeric quantity and suggested/recent/custom unit | Families measure by mass, jar, frame, or super |
| Follow-up | Recent/custom description and date | Repeated chores benefit from recent values without a rigid taxonomy |
| Names, locations, access, origin, notes | Free text with length validation | These values are inherently local and user-defined |

Suggestions are rendered as native datalists: keyboard and assistive-technology
users can choose a listed value or type a custom value in the same labeled control.
Recent values are organization-scoped records already synchronized to the device;
they do not leave the user's deployment for suggestion generation.
