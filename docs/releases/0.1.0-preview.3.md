# ApiaryLens 0.1.0 Preview 3

Released: 2026-07-17

ApiaryLens Preview 3 is a **Public Preview**, not GA or a stable release. It is the
current Windows-first and family-connected evaluation build. Features, contracts,
and documentation may continue to change quickly during Preview; keep verified
backups and exports, and do not make Preview the only copy of irreplaceable hive
records or media.

## What changed

- Added the production standalone-to-connected Windows migration workflow: verified
  backup, secret-free SQLite journal, bounded exact-once record/tombstone/media
  transfer, interruption resume, full hash reconciliation, atomic authority cutover,
  and rollback only while the connected family has no newer remote-only writes.
- Added an accessible bottom-to-top hive equipment stack with add/remove/reorder,
  custom component types and purposes, installed/removal history, offline saves, and
  normal synchronization.
- Made inspection weather an explicit optional offline snapshot with observation and
  provenance fields plus a consent-gated provider-neutral enrichment boundary. No
  external weather service is required or enabled.
- Changed server sync batches to commit domain writes and idempotency receipts in one
  bounded SQLite transaction.
- Completed exact released Product Preview 2 lifecycle acceptance on disposable
  Cloudflare and clean Ubuntu targets, including update, repair, restore, rollback,
  uninstall/reinstall, private media, and complete cleanup.
- Completed the safely automatable installed-Windows acceptance rows: installer
  identity, loopback/security boundary, backup/restore, keep-data reinstall, and
  automated accessibility/reflow profiles.

## Windows Preview warning

The Preview 3 Windows Setup is intentionally published as
`ApiaryLensSetup-UNSIGNED-PREVIEW.exe`. It is not Authenticode signed. This exception
applies only to the explicitly labeled Preview installer; RC and Stable remain
signed-only. Verify the published SHA-256 and GitHub artifact attestation before
testing.

## Compatibility

- Product: `0.1.0-preview.3`
- API contract: `1.0`
- Sync contract: `1`
- Database migration head: `0004`
- Deployment-plan contract: `1`
- Compatible installer: Scout Bee `0.1.0-preview.4`

Scout Bee and ApiaryLens are independently versioned. Scout Preview 4 supports the
exact Product Preview 3 identity and continues to require explicit Advanced opt-in
for Preview channels.

## Still open before GA

- Production Authenticode signing and signed installer acceptance.
- Physical iPhone/iPad installed-PWA offline/update acceptance.
- Manual NVDA, VoiceOver, retail Windows High Contrast, and native Electron zoom
  verification.
- Project-owner release signoff.

These gates do not prevent use of the explicitly labeled Public Preview, but they do
prevent a Stable or GA claim.
