# ApiaryLens 0.1.0 Preview 3

Development record: 2026-07-17

This file is retained as an internal development record. Preview 3 is not the
current public release and is not a supported installation path. ApiaryLens Public
Preview 1 is the current release. The items below describe experimental work that
may inform a future Preview.

## What changed

- Added the production standalone-to-connected Windows migration workflow: verified
  backup, secret-free SQLite journal, bounded exact-once record/tombstone/media
  transfer, interruption resume, full hash reconciliation, atomic authority cutover,
  and rollback only while the connected family has no newer remote-only writes.
- Added an accessible bottom-to-top hive equipment stack with add/remove/reorder,
  custom component types and purposes, installed/removal history, offline saves, and
  normal synchronization.
- Made inspection weather an explicit optional offline snapshot with observation and
  provenance fields plus a consent-gated provider-neutral enrichment boundary. No
  external weather service is required or enabled.
- Changed server sync batches to commit domain writes and idempotency receipts in one
  bounded SQLite transaction.
- Completed exact released Product Preview 2 lifecycle acceptance on disposable
  Cloudflare and clean Ubuntu targets, including update, repair, restore, rollback,
  uninstall/reinstall, private media, and complete cleanup.
- Completed the safely automatable installed-Windows acceptance rows: installer
  identity, loopback/security boundary, backup/restore, keep-data reinstall, and
  automated accessibility/reflow profiles.
- Completed exact installed Preview 3 migration acceptance with 527 records, a
  tombstone, private media, six bounded batches, interruption/resume, exact-once
  reconciliation, authority cutover, rollback, and secret-free journal evidence.

## Experimental Windows build

An unsigned Windows setup executable was produced during engineering validation. It
was withdrawn from end-user distribution and is not an ApiaryLens release. The
experiment does not change the Public Preview 1 support boundary: no Windows desktop
package is currently offered.
