# ApiaryLens 0.1.0 Preview 2

## Release status

ApiaryLens Preview 2 is a **Public Preview**, not GA or a stable release. It is an
immutable follow-up to Preview 1 for real family, Windows, Cloudflare, and
self-hosted evaluation. Interfaces and workflows may continue to change quickly.
Keep verified backups and exports; do not use a Preview as the only copy of
irreplaceable apiary records or original photos.

## What changed

- Added the standalone Windows application with current-user Setup, local SQLite
  records, private filesystem media, protected local-service control, offline use,
  verified `.albackup` backup, staged restore, session revocation, health checks,
  recovery rollback, and keep-data uninstall.
- Made connected synchronization automatic on open, resume, reconnect, and online
  saves, including multi-batch push, paginated pull, organization-scoped cursors,
  and protection for pending or conflicted local work.
- Corrected installed-PWA offline relaunch and local-session hydration so cached
  navigation and records render before a stalled network refresh.
- Hardened offline photo staging, display, status, upload, retry, and synchronized
  media behavior.
- Added clickable Overview destinations, intelligent field controls, queen marking
  color/year choices, a searchable beekeeping glossary, manual inspection weather,
  family member administration, update notices, session transparency, and hive
  equipment visualization foundations.
- Added a secret-free `includeWebFrontend` deployment choice. Cloudflare and remote
  Linux may run the backend alone or the backend plus PWA while preserving API,
  authentication, sync, private media, HTTPS, health, backup, and recovery.
- Strengthened Windows-to-Linux orchestration tests for pinned SSH identity,
  argument-safe SCP/SSH, protected runtime-secret streams, exact build-health
  identity, and host-key-change rejection.
- Published separately versioned Scout Bee lifecycle packages and complete Windows,
  Linux, Cloudflare, backup, recovery, diagnostics, and CI handoff guidance.

## Install and verify

Use the [public installation hub](https://apiarylens.org/install/) for the exact
Windows installer, Scout Bee package, published SHA-256 values, release manifests,
and platform guidance. Product deployment manifests and immutable Cloudflare and
Compose bundles publish under
`https://apiarylens.org/releases/0.1.0-preview.2/`.

The Preview Windows installer is explicitly unsigned until production
Authenticode material is available. Windows may show **Windows protected your PC**.
Use **More info → Run anyway** only after the downloaded file matches the SHA-256
published by the immutable release.

## Known acceptance gates

Preview 2 publication does not claim GA. The remaining gates are maintained in the
[MVP UAT record](../testing/mvp-uat.md) and include exact released-byte Cloudflare
and clean Linux lifecycle replay, interrupted production migration, physical
iPhone/iPad offline and reconnect acceptance, physical Windows accessibility and
recovery task tests, and production signing.

## Feedback and support

Report reproducible defects with the exact product and Scout versions, deployment
profile, browser or device, and sanitized diagnostics. Never include passwords,
tokens, recovery codes, private locations, hive records, or original media in a
public issue. Follow [SECURITY.md](../../SECURITY.md) for security or data-loss
reports.
