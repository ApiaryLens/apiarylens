# Task 004: Define Data Model

## Status

MVP logical model completed 2026-07-15. Physical schema and migrations are verified
as part of implementation.

The accepted model is documented in
[MVP Data Model](../docs/architecture/data-model.md) and ADR 0009. MVP entities are:

- Organization
- User
- Membership and Role
- Session, Invitation, and Recovery Code
- Apiary
- Hive
- Equipment Box
- Queen
- Inspection
- Media Asset
- Mite Count
- Health Observation
- Treatment Event
- Feeding Event
- Harvest
- Follow-up Item
- Change Log and Idempotency Record
- Audit Event

The Lucid ERD and accessible PNG export are cataloged under `docs/diagrams/`.
Frames, AI review, live providers, public sharing, clubs, research, galleries,
registries, and commercial extensions remain compatible post-MVP concepts.
